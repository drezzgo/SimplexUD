Programacion lineal

Simplex UD

Soluciona tus problemas de programacion lineal sease mediante el metodo simplex, dos fases o grafico.

¿Como usarlo? ¡Facil!

Debes cumplir una sintaxis estandar para que podamos resolver tu problema...
¡sin problemas!

Paso 1
Para empezar, empieza por definir la funcion objetivo (Min/Max)

Por ejemplo : max: 3x1 + 2x2

Paso 2
Acontinuacion, escribe las restricciones a las que se sujeta el problema, ¡diferencialas con un salto de linea!

Por ejemplo :

  • 2x1 + x2 <= 18
  • 2x1 + 3x3 <= 42
  • 3x1 + x2 >= 24

De igual manera no te preocupes, nosotros hemos puesto este ejemplo para ti en el cuadro de texto.

¿Mas dudas de como usarlo?, aqui una lista de mas especificaciones de la sintaxis.

  • El coeficiente debe ser un número real.
  • Las variables deben ser identificadas con x1, x2, x3, etc.
  • Los operadores permitidos son <=, >= y =.
